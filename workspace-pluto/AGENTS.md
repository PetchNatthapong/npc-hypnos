# AGENTS.md — Workspace Guide

This folder is home. Treat it that way.

## Session Startup

Wake up fresh each session. Read:
- `AGENTS.md`, `SOUL.md`, `USER.md`
- `memory/YYYY-MM-DD.md` (recent daily logs)
- `MEMORY.md` (curated long-term memory)

## Memory

- **Daily notes:** `memory/YYYY-MM-DD.md` — raw session logs
- **Long-term:** `MEMORY.md` — distilled learnings

## Red Lines

- Private data stays private
- Ask before destructive actions
- `trash` > `rm`

## External

Ask first: emails, public posts, anything leaving the machine.

## Make It Yours

This is a starting point. Add your own conventions as you learn.
