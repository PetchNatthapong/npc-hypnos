## 2️⃣ Physical Examination

**Vital Signs:**
| | |
|---|---|
| T | 36.8°C (ไม่มีไข้) |
| P | 82/min |
| BP | 130/85 mmHg |
| RR | 20/min |
| SpO₂ | **92% on RA** ⚠️ |

**Skin:**
- **Gottron papules:** ผื่นแดงบุ๋มบริเวณ dorsal PIP joints ทั้งสองข้าง
- **Heliotrope rash:** ผื่นสีม่วงแดงรอบตา (periorbital) ทั้งสองข้าง
- **V-sign:** ผื่นแดงบริเวณหน้าอกด้านหน้า (ข้างหลังคอ)
- **Shawl sign:** ผื่นแดงบริเวณหลังส่วนบน
- **Mechanic's hands:** ผิวหนังนิ้วมือหยาบดำทั้งสองข้าง
- ไม่มี skin sclerosis

**Musculoskeletal:**
- **Motor power:** Grade 3/5 ( proximal muscles ทั้งสองข้าง — ยกแขนเหนือศีรษะไม่ได้)
- **Distal power:** Grade 4/5 (ปกติ)
- **Reflexes:** ปกติ symmetrical
- **Gower's sign:** Positive (ต้องใช้มือช่วยจากพื้น)
- **Muscle tenderness:** มีเล็กน้อยที่ต้นแขน

**Respiratory:**
- ฟังปอด: **fine bibasilar crackles** ฐานปอดทั้งสองข้าง
- ไม่มี wheezing

**Cardiovascular:**
- ฟังปกติ, no murmur
- JVP ไม่สูง

**GI:**
- กลืนลำบาก — ตรวจ swallowing test: มี residual เล็กน้อย
